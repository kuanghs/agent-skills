# 中国广告法合规审查器

## 概述

对广告文案、营销素材、商品描述进行《广告法》合规审查，识别违禁词汇、违法表述和品类特殊限制，输出结构化审查报告。

## 审查流程

### 第一步：违禁词汇筛查

对照 `references/prohibited-words-catalog.md` 检查文案中的违禁词汇。

**九大类违禁词：**

| 类别 | 典型违禁词 | 法律依据 |
|------|-----------|---------|
| 绝对化用语 | 最、第一、唯一、顶级、极品 | 《广告法》第9条第3项 |
| 权威性暗示 | 国家级、特供、专供、免检 | 《广告法》第9条第2项 |
| 虚假承诺 | 包治百病、零风险、100%有效 | 《广告法》第4条、第28条 |
| 诱导性表述 | 限时仅剩、立即抢购（无真实依据） | 《广告法》第28条 |
| 迷信内容 | 带来好运、辟邪、风水 | 《广告法》第9条第7项 |
| 歧视性内容 | 暗示性别/年龄/地域歧视 | 《广告法》第9条第6项 |
| 损害尊严 | 妨碍环境/自然资源保护 | 《广告法》第9条第5项 |
| 涉及专利未标号 | 专利技术（未标明专利号和种类） | 《广告法》第12条 |
| 伪造荣誉 | 虚假获奖、伪造认证 | 《广告法》第28条 |

### 第二步：品类特殊限制审查

不同品类有额外的广告限制：

| 品类 | 关键限制 | 法律依据 |
|------|---------|---------|
| **医疗/药品** | 不得含治愈率/有效率保证；不得与其他药品/医疗器械功效比较；不得利用广告代言人推荐；处方药只能在指定医学药学专业刊物广告；非处方药须标明"请按药品说明书或者在药师指导下购买和使用" | 《广告法》第15-16条 |
| **保健食品** | 不得表示治疗功能；不得与药品比较；需标注"本品不能代替药物" | 《广告法》第18条 |
| **医疗器械** | 需经审查批准；不得含治愈率/有效率承诺；推荐个人自用须标明"请仔细阅读产品说明书或者在医务人员的指导下购买和使用" | 《广告法》第16条 |
| **金融/理财** | 不得含收益保证性承诺；明示或暗示保本/无风险/保收益均禁止；需提示风险；不得利用学术机构/专业人士/受益者名义推荐 | 《广告法》第25条 |
| **教育培训** | 不得含升学/通过考试/获得学位学历的保证性承诺；不得暗示考试机构/命题人员参与；不得利用受益者名义推荐 | 《广告法》第24条 |
| **房地产** | 不得含升值/投资回报承诺；需标明建筑面积/套内面积 | 《广告法》第26条 |
| **食品** | 不得含预防/治疗功能；不得与药品比较 | 《食品安全法》第71条 |
| **化妆品** | 不得含医疗效果；不得暗示治疗功能 | 《化妆品监督管理条例》 |
| **酒类** | 不得诱导/怂恿饮酒；不得出现饮酒动作；不得表现驾驶活动；不得暗示饮酒消除紧张/增加体力 | 《广告法》第23条 |
| **烟草** | 禁止在大众媒体发布烟草广告 | 《广告法》第22条 |
| **母婴/婴幼儿** | 不得替代母乳宣传；奶粉不得声称优于母乳 | 《广告法》第20条 |

### 第三步：广告形式合规审查

| 检查项 | 要求 | 法律依据 |
|--------|------|---------|
| 广告标识 | 需使消费者辨明为广告 | 《广告法》第14条 |
| 代言人/推荐人 | 不得为其未使用过的商品作推荐；医疗/保健/金融不得利用代言人；不满十周岁未成年人不得作代言人 | 《广告法》第38条 |
| 比较广告 | 不得贬低他人产品；不得使用国家级/行业级数据未标明来源 | 《广告法》第13条 |
| 引用数据 | 需标明出处；不得伪造 | 《广告法》第11条 |
| 未成年人保护 | 不得在校园/教科书内广告；不得诱导家长购买 | 《广告法》第39-40条 |
| 直播带货 | 需标明广告性质；主播需真实使用体验 | 《网络直播营销管理办法》 |
| 种草/软文 | 需标明"广告"或"推广" | 《互联网广告管理办法》第9条 |

### 第四步：电商平台特殊要求

| 平台 | 常见违规 | 处罚 |
|------|---------|------|
| 淘宝/天猫 | 极限词、虚假宣传、盗图 | 扣分、下架、罚款 |
| 京东 | 极限词、虚假比较、价格欺诈 | 下架、扣分 |
| 抖音/快手 | 虚假功效、违禁品类、诱导交易 | 封号、限流 |
| 小红书 | 虚假种草、未标广告、夸大效果 | 限流、封号 |
| 拼多多 | 虚假折扣、极限词、低俗营销 | 下架、罚款 |

## 审查报告格式

```markdown
[广告法合规审查报告]

审查对象：{广告文案/商品描述/营销素材}
审查时间：{YYYY-MM-DD HH:mm}

第一步（违禁词筛查）：
  - 发现违禁词：{列表，含位置和类别}
  - 风险等级：{🟢无 / 🟡低 / 🔴高}

第二步（品类限制审查）：
  - 品类：{医疗/保健/金融/食品/化妆品/...}
  - 合规状态：{✅通过 / ⚠️需修改 / ❌不合规}
  - 具体问题：{详情}

第三步（形式合规审查）：
  - 广告标识：{合规/缺失}
  - 代言人合规：{合规/不合规}
  - 数据引用：{合规/需标明出处}
  - 未成年人保护：{合规/需修改}

第四步（平台要求）：
  - 目标平台：{平台名}
  - 合规状态：{✅通过 / ⚠️需修改}

总体结论：{✅可发布 / ⚠️修改后可发布 / ❌不可发布}
修改建议：{具体修改方案，按优先级排序}
```

## 使用示例

**示例1：电商商品标题"全网最低价！国家级认证美白神器"**
- 第一步：命中"最低价"（绝对化用语）+ "国家级"（权威性暗示）→ 🔴
- 第二步："美白"涉及化妆品，不得暗示医疗效果 → ⚠️
- 第三步：无广告标识 → ⚠️
- 结论：❌不可发布，需删除"全网最低价""国家级"，修改"神器"表述

**示例2：保健食品广告"XX胶囊，降血糖效果显著，李医生推荐"**
- 第一步：无绝对化用语 → 🟢
- 第二步：保健食品不得表示治疗功能（"降血糖"）+ 不得利用医师名义推荐 → 🔴
- 第三步：缺少"本品不能代替药物"标注 → 🔴
- 结论：❌不可发布，需删除治疗功能表述和医师推荐，添加强制声明

**示例3：普通商品描述"舒适透气的纯棉T恤，日常休闲好选择"**
- 第一步：无违禁词 → 🟢
- 第二步：普通服装，无特殊限制 → ✅
- 第三步：无代言人/数据引用问题 → ✅
- 结论：✅可发布
