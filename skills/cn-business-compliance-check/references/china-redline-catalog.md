# 中国法律红线清单

> 本清单列出10大绝对禁止领域，每个领域附带具体法条、处罚标准、典型场景。
> 命中任一项 → 立即终止操作，输出合规预警报告。

## 1. 非法集资与传销

### 非法吸收公众存款罪

- **法条**：《刑法》第176条
- **处罚**：
  - 数额较大（个人20万以上/单位100万以上）：3年以下有期徒刑或拘役，并处2-20万罚金
  - 数额巨大（个人100万以上/单位500万以上）或有其他严重情节：3-10年有期徒刑，并处5-50万罚金
  - 数额特别巨大：10年以上有期徒刑，并处罚金
- **典型场景**：承诺年化收益15%向社会公众吸收资金，未经银保监会批准
- **关键特征**：承诺高额回报 + 向社会不特定对象吸收资金 + 未经批准

### 集资诈骗罪

- **法条**：《刑法》第192条
- **处罚**：
  - 数额较大：5年以下有期徒刑或拘役，并处2-20万罚金
  - 数额巨大：5-10年有期徒刑，并处5-50万罚金
  - 数额特别巨大：10年以上有期徒刑或无期徒刑，并处5-50万罚金或没收财产
- **典型场景**：虚构项目以高回报为诱饵骗取资金，资金实际用于个人挥霍或还债

### 组织、领导传销活动罪

- **法条**：《刑法》第224条之一
- **处罚**：
  - 组织者/领导者：5年以下有期徒刑或拘役，并处罚金
  - 情节严重：5年以上有期徒刑，并处罚金
- **典型场景**：要求缴纳入门费→拉人头发展下线→按层级计酬（三级以上即构成）
- **关键特征**：入门费 + 拉人头 + 团队计酬（三级以上）
- **法规**：《禁止传销条例》（国务院令第444号）

### 防范和处置非法集资条例

- **法规**：《防范和处置非法集资条例》（2021年5月1日施行）
- **关键条款**：
  - 第19条：任何单位和个人不得从非法集资中获取经济利益
  - 第25条：因参与非法集资受到的损失，由集资参与人自行承担

## 2. 金融违法

### 内幕交易罪

- **法条**：《刑法》第180条 + 《证券法》第53条
- **处罚**：
  - 情节严重：5年以下有期徒刑或拘役，并处或单处违法所得1-5倍罚金
  - 情节特别严重：5-10年有期徒刑，并处违法所得1-5倍罚金
- **典型场景**：利用未公开的重大并购信息提前买入股票获利

### 操纵证券、期货市场罪

- **法条**：《刑法》第182条 + 《证券法》第55条
- **处罚**：
  - 情节严重：5年以下有期徒刑或拘役，并处或单处罚金
  - 情节特别严重：5年以上有期徒刑，并处罚金
- **典型场景**：联合多个账户集中买入推高股价后抛售（"拉高出货"）

### 洗钱罪

- **法条**：《刑法》第191条 + 《反洗钱法》
- **处罚**：
  - 5年以下有期徒刑或拘役，并处或单处洗钱数额5-20%罚金
  - 情节严重：5-10年有期徒刑，并处洗钱数额5-20%罚金
- **典型场景**：将毒品犯罪所得通过虚拟货币交易转移至境外账户

### 非法经营罪（金融业务）

- **法条**：《刑法》第225条
- **处罚**：
  - 情节严重：5年以下有期徒刑或拘役，并处或单处违法所得1-5倍罚金
  - 情节特别严重：5年以上有期徒刑，并处违法所得1-5倍罚金或没收财产
- **典型场景**：无牌照开展支付结算、资金借贷、保险、基金销售等业务

## 3. 网络犯罪

### 非法侵入计算机信息系统罪

- **法条**：《刑法》第285条第1款
- **处罚**：3年以下有期徒刑或拘役
- **侵入国家事务、国防建设、尖端科学技术领域计算机信息系统**：直接入罪，不要求后果

### 非法获取计算机信息系统数据罪

- **法条**：《刑法》第285条第2款
- **处罚**：
  - 情节严重：3年以下有期徒刑或拘役，并处或单处罚金
  - 情节特别严重：3-7年有期徒刑，并处罚金
- **典型场景**：撞库获取他人账号密码、SQL注入获取数据库信息

### 提供入侵工具罪

- **法条**：《刑法》第285条第3款
- **处罚**：
  - 情节严重：3年以下有期徒刑或拘役，并处或单处罚金
  - 情节特别严重：3-7年有期徒刑，并处罚金
- **典型场景**：制作/售卖黑客工具、爬虫软件（专门用于侵入系统）

### 破坏计算机信息系统罪

- **法条**：《刑法》第286条
- **处罚**：
  - 情节严重：5年以下有期徒刑或拘役
  - 情节特别严重：5年以上有期徒刑
- **典型场景**：DDoS攻击、删除/修改系统数据、恶意软件破坏系统运行

### 拒不履行信息网络安全管理义务罪

- **法条**：《刑法》第286条之一
- **处罚**：
  - 情节严重：3年以下有期徒刑、拘役或管制，并处或单处罚金
- **典型场景**：网络平台拒不整改违法信息、拒不提供技术支持协助侦查

## 4. 侵犯公民个人信息

### 侵犯公民个人信息罪

- **法条**：《刑法》第253条之一 + 《个人信息保护法》(2021) + 《数据安全法》(2021) + 《网络安全法》第44条
- **处罚**：
  - 情节严重：3年以下有期徒刑或拘役，并处或单处罚金
  - 情节特别严重：3-7年有期徒刑，并处罚金
- **入罪标准**：
  - 非法获取/出售一般个人信息50条以上
  - 非法获取/出售敏感个人信息（行踪轨迹、通信内容、征信信息、财产信息）**25条以上**
  - 非法获取/出售住宿信息、通信记录、健康生理信息、交易信息**500条以上**
  - 违法所得5000元以上
- **典型场景**：未经授权爬取用户数据出售、购买他人身份证信息批量注册账号
- **关键红线**：50条敏感信息即入罪，无需造成实际损害

### 个人信息保护法关键条款

- **第13条**：处理个人信息需取得个人同意（7种例外情形）
- **第28条**：处理敏感个人信息需取得单独同意
- **第66条**：违法处理个人信息，罚款100万元以下；情节严重，罚款5000万元以下或上一年度营业额5%

## 5. 知识产权侵权

### 侵犯著作权罪

- **法条**：《刑法》第217条 + 《著作权法》
- **处罚**：
  - 违法所得数额较大（3万元以上）或有其他严重情节：3年以下有期徒刑或拘役，并处或单处罚金
  - 违法所得数额巨大（15万元以上）或有其他特别严重情节：3-10年有期徒刑，并处罚金
- **典型场景**：批量盗版软件/影视内容分发、未经授权复制销售他人作品

### 假冒注册商标罪

- **法条**：《刑法》第213-215条 + 《商标法》
- **处罚**：
  - 情节严重（非法经营数额5万元以上）：3年以下有期徒刑或拘役，并处或单处罚金
  - 情节特别严重（非法经营数额25万元以上）：3-7年有期徒刑，并处罚金
- **典型场景**：在同类商品上使用与他人注册商标相同的标识

### 假冒专利罪

- **法条**：《刑法》第216条 + 《专利法》
- **处罚**：3年以下有期徒刑或拘役，并处或单处罚金

### 侵犯商业秘密罪

- **法条**：《刑法》第219条 + 《反不正当竞争法》第9条
- **处罚**：
  - 情节严重：3年以下有期徒刑或拘役，并处或单处罚金
  - 情节特别严重：3-10年有期徒刑，并处罚金
- **典型场景**：窃取/泄露竞争对手技术秘密、客户名单

## 6. 黄赌毒

### 制作、贩卖、传播淫秽物品罪

- **法条**：《刑法》第363-367条
- **处罚**：
  - 制作/复制/出版/贩卖/传播：3年以下有期徒刑、拘役或管制，并处罚金
  - 情节严重：3-10年有期徒刑，并处罚金
  - 情节特别严重：10年以上有期徒刑或无期徒刑，并处罚金或没收财产
- **治安处罚**：《治安管理处罚法》第68条——制作/传播淫秽信息，处15日以下拘留+3000元以下罚款

### 赌博罪 / 开设赌场罪

- **法条**：《刑法》第303条
- **处罚**：
  - 赌博罪：3年以下有期徒刑、拘役或管制，并处罚金（以营利为目的聚众赌博）
  - 开设赌场罪：5年以下有期徒刑、拘役或管制，并处罚金；情节严重5-10年
- **注意**：网络赌博同样入罪。开设线上赌场、组织网络赌局均属开设赌场罪
- **治安处罚**：《治安管理处罚法》第70条——参与赌博，处15日以下拘留+3000元以下罚款

### 毒品犯罪

- **法条**：《刑法》第347-357条
- **处罚**：
  - 走私/贩卖/运输/制造鸦片1000克以上、海洛因/冰毒50克以上：15年有期徒刑、无期徒刑或死刑，并处没收财产
  - 数量较少：7年以上有期徒刑，并处罚金
  - 少量（鸦片200克以下、海洛因/冰毒10克以下）：3年以下有期徒刑、拘役或管制，并处罚金
- **注意**：持有毒品也构成犯罪（《刑法》第348条）

## 7. 诈骗与虚假宣传

### 诈骗罪

- **法条**：《刑法》第266条
- **处罚**：
  - 数额较大（3000元以上）：3年以下有期徒刑、拘役或管制，并处或单处罚金
  - 数额巨大（3万元以上）或有其他严重情节：3-10年有期徒刑，并处罚金
  - 数额特别巨大（50万元以上）或有其他特别严重情节：10年以上有期徒刑或无期徒刑，并处罚金或没收财产

### 虚假广告罪

- **法条**：《刑法》第222条 + 《广告法》第28条
- **处罚**：2年以下有期徒刑或拘役，并处或单处罚金
- **主体**：广告主、广告经营者、广告发布者（明知虚假仍设计/制作/发布）

### 刷单炒信

- **法规**：《反不正当竞争法》第8条 + 《电子商务法》第17条
- **处罚**：
  - 行政处罚：20万元以上100万元以下罚款；情节严重100万元以上200万元以下罚款，可吊销营业执照
  - 刑事风险：组织刷单可能构成非法经营罪（《刑法》第225条）
- **典型场景**：花钱雇人刷好评、虚假交易制造销量数据

### 招摇撞骗罪

- **法条**：《刑法》第279条
- **处罚**：3年以下有期徒刑、拘役、管制或剥夺政治权利；情节严重3-10年
- **典型场景**：冒充国家机关工作人员身份进行诈骗

### 伪造国家机关公文、证件、印章罪

- **法条**：《刑法》第280条
- **处罚**：3年以下有期徒刑、拘役、管制或剥夺政治权利；情节严重3-10年
- **典型场景**：伪造营业执照、资质证书、检测报告

## 8. 武器与危险品

### 非法制造、买卖、运输、储存枪支弹药爆炸物罪

- **法条**：《刑法》第125-130条
- **处罚**：
  - 第125条：3年以上10年以下有期徒刑；情节严重10年以上有期徒刑、无期徒刑或死刑
  - 第128条（非法持有/私藏枪支）：3年以下有期徒刑、拘役或管制；情节严重3-7年
- **法规**：《枪支管理法》 + 《危险化学品安全管理条例》

## 9. 人口与器官交易

### 拐卖妇女、儿童罪

- **法条**：《刑法》第240条
- **处罚**：5年以上10年以下有期徒刑，并处罚金；有8种加重情节之一10年以上有期徒刑或无期徒刑，并处罚金或没收财产；情节特别严重死刑
- **加重情节**：拐卖3人以上、造成被拐卖人重伤/死亡、奸淫被拐卖人等

### 收买被拐卖的妇女、儿童罪

- **法条**：《刑法》第241条
- **处罚**：3年以下有期徒刑、拘役或管制
- **注意**：2015年《刑法修正案(九)》取消"可以不追究刑事责任"条款，一律追究

### 组织出卖人体器官罪

- **法条**：《刑法》第234条之一
- **处罚**：5年以下有期徒刑，并处罚金；情节严重5年以上有期徒刑，并处罚金或没收财产
- **法规**：《人体器官移植条例》——任何组织/个人不得以任何形式买卖人体器官

## 10. 逃税与虚假财务

### 逃税罪

- **法条**：《刑法》第201条 + 《税收征收管理法》第63条
- **处罚**：
  - 逃避缴纳税款数额较大（5万元以上）且占应纳税额10%以上：3年以下有期徒刑或拘役，并处罚金
  - 数额巨大（25万元以上）且占应纳税额30%以上：3-7年有期徒刑，并处罚金
- **初犯免罪条款**：经税务机关依法下达追缴通知后，补缴应纳税款、缴纳滞纳金、已受行政处罚的，不予追究刑事责任（5年内因逃税受过刑事处罚或被税务机关给予二次以上行政处罚的除外）

### 虚开增值税专用发票罪

- **法条**：《刑法》第205条
- **处罚**：
  - 3年以下有期徒刑或拘役，并处2-20万罚金
  - 数额较大或有其他严重情节：3-10年有期徒刑，并处5-50万罚金
  - 数额巨大或有其他特别严重情节：10年以上有期徒刑或无期徒刑，并处5-50万罚金或没收财产

### 虚开发票罪

- **法条**：《刑法》第205条之一
- **处罚**：
  - 情节严重：2年以下有期徒刑、拘役或管制，并处罚金
  - 情节特别严重：2-7年有期徒刑，并处罚金

### 违规披露、不披露重要信息罪

- **法条**：《刑法》第161条
- **处罚**：3年以下有期徒刑或拘役，并处或单处2-20万罚金
- **主体**：公司/企业依法负有信息披露义务的直接责任人员
