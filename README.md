# Agent Skills

中国法律合规审查 AI Skills，适用于 OpenClaw / ClawHub 智能体。

## 包含的 Skills

| Skill | 说明 | 文档 |
|-------|------|------|
| **cn-business-compliance-check** | 中国商业合规审查 — 三层过滤机制、317条法条+处罚标准、灰色地带指南 | [SKILL.md](skills/cn-business-compliance-check/SKILL.md) |
| **cn-advertising-compliance** | 广告法合规审查 — 14类违禁词库、替代话术、处罚案例匹配 | [SKILL.md](skills/cn-advertising-compliance/SKILL.md) |
| **cn-ecommerce-compliance** | 跨境电商合规 — 产品认证指南、跨境税务(VAT/GST)、平台规则 | [SKILL.md](skills/cn-ecommerce-compliance/SKILL.md) |

## 安装

通过 ClawHub 安装：

```bash
clawhub install @kuanghs/cn-business-compliance-check
clawhub install @kuanghs/cn-advertising-compliance
clawhub install @kuanghs/cn-ecommerce-compliance
```

## 在线工具

[liaohegui.top](https://liaohegui.top) — 输入文案，3秒出合规审查报告。
